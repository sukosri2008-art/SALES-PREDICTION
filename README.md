# Sales Prediction

## Objective
Predict product sales using advertising budget data and machine learning techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Dataset Features
- TV Ad Budget
- Radio Ad Budget
- Newspaper Ad Budget
- Sales

## Model Used
- Linear Regression

## Workflow
1. Data Loading
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Sales Prediction
6. Performance Evaluation

## Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)

## Result
The model achieved an R² Score of approximately 0.90, indicating strong predictive performance.

## Conclusion
Advertising expenditure has a significant impact on sales and can be used to forecast future sales effectively..
